# Convert a PACTA style quarter string to an IMF style quarter string

Convert a PACTA style quarter string to an IMF style quarter string

## Usage

``` r
quarter_to_imf_timestamp(quarter)
```

## Arguments

- quarter:

  A character vector containing PACTA style quarter strings in the form
  e.g. "2021Q4"X

## Value

A character vector containing the equivalent IMF style quarter strings
e.g. "2021-Q4"
